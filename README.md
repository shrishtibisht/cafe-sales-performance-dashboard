# Cafe Sales Performance Dashboard

## Project Overview

This project analyzes cafe sales data using Excel and Power BI to identify revenue trends, product profitability, and business improvement opportunities through interactive dashboards.

## Tools Used

* Excel (data cleaning, pivot tables)
* Power BI (dashboard development)
* KPI analysis and business insights

## Dataset Workflow

Raw sales data was cleaned in Excel and transformed using pivot tables before building an interactive Power BI dashboard for performance tracking.

## Key KPIs Analyzed

* Total Revenue
* Total Profit
* Profit Margin
* Product-level performance
* Monthly sales trends

## Key Insights

* Cappuccino and Latte generated the highest profit margins
* Pasta and Sandwiches produced high revenue but lower margins
* Seasonal fluctuations impacted monthly revenue consistency

## Recommendations

* Promote high-margin beverages through combo offers
* Optimize pricing strategy for low-margin food items
* Introduce bundled promotions to increase average order value
